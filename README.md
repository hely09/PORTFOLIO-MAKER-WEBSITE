🌐 Portfolio Maker Website

A Flask-based web application that allows users to create and customize professional portfolio websites using multiple templates.

🚀 Features
Multiple portfolio design templates (4 designs)
User-friendly form to enter personal details
Image upload functionality for profile pictures
Dynamic portfolio generation using templates
Responsive UI using Bootstrap
Social media integration (GitHub, Instagram)
🛠️ Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript, Bootstrap
Templating Engine: Jinja2

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/hely09/portfolio-maker.git
Navigate to the project folder:
cd portfolio-maker
Install dependencies:
pip install -r requirements.txt
Run the application:
python app.py
Open in browser:
http://127.0.0.1:5000/
